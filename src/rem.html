# Understanding rem and em
1. What is rem - Root em
Rem is the relative unit that depend on the font-size which define in the html tag
2. Diffirent between rem and em
em depend on font-size of that element
For example: that element covered by em-class
html { font-size: 16px; }
.em-class { font-size: 20px; padding: 1em; border: 1px dashed red; }
Every em in this element that depend on font-size of this block 20px
